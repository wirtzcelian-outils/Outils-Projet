# Documentation Composants

Ce dossier contient les composants d'interface utilisateur (UI) réutilisables dans différentes pages de l'application.

## Fichiers

*   **`Navbar.jsx`** : Barre de navigation supérieure. Affiche le logo du site, le nom de l'utilisateur connecté et le bouton de déconnexion.
*   **`SortableItem.jsx`** : Composant représentant un film dans une liste triable. Il gère l'affichage du film, et permet le Drag & Drop (via `@dnd-kit`), la modification de commentaires et la suppression de l'élément.
*   **`SplashScreen.jsx`** : Écran de chargement affiché au démarrage de l'application pendant la vérification de l'état du serveur backend.
